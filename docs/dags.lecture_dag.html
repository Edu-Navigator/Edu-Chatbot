dags.lecture_dag module

데이터 수집 DAG들이 성공으로 종료되면, 원본 데이터를 전처리하고 취합 및 적재한다.

이 DAG는 데이터베이스에서 각 테이블의 컬럼명을 표준화하여 타켓 테이블의 스키마에 맞게 변환, 강의장 주소 기준 행정구역(시, 도, 구) 표준화, 강의 이미지 URL 맵핑 등의 처리를 수행하고 적재합니다.

param schedule_interval:

매일 오전 10시 20분 (‘20 10 * * *’)

type schedule_interval:

str

param start_date:

Starting date for the DAG (2025-12-01)

type start_date:

datetime

param catchup:

False to prevent backfilling historical runs

type catchup:

bool

param tags:

[“02”, “postgres”, “lecture”, ‘edu_info’]

type tags:

list of str
